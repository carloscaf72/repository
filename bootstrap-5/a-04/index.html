Breakpoints Responsivos

  • A principal característica do Bootstrap é o layout responsivo baseado no design Mobile First que significa que a criação de página deve ser pensada para dispositivos móveis (celulares), com baixa resolução de largura de tela. Para isso ele se utiliza dos Breakpoints.
  • Os Breakpoints são infixos adicionados as demais classes do Bootstrap e representam algumas possíveis larguras de tela comumente encontradas em diferentes dispositivos que acessam a WEB e são usados para se definir como será o design da página para cada resolução (largura) de tela.
  • A tabela abaixo demonstra os seis (6) tamanhos de resolução e os Infixos que representam as classes de breakpoints para cada uma, onde será possível alterar o layout da página em cada uma dessas resoluções especificas.

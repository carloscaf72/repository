Cards

  • O Bootstrap contém diversas classes para criação e configurações de cards, melhorando seu visual em relação ao HTML padrão.

cards:

  • Para a criação de um card utilizamos um elemento de bloco (div) com a classe card.
  • Por padrão, o componente ocupará 100% da largura do elemento pai.

<div>

(card)

card-body:

card-title / card-subtitle / card-text

  • A classe card-body será o elemento de bloco (div) do corpo do card onde são colocados os componentes de título com a classe card-title, o subtítulo com a classe card-subtitle e os textos descritivos com a classe card-text.
  • As classes card-title e card-subtitle não alteram a aparência do componente, devendo assim ser complementadas com elementos ou classes para formatação do tamanho e/ou aparência.

<div class="card-body">

(card-title/card-subtitle/card-text)

Campos de Checagem

  • O Bootstrap contém diversas classes para formatar formulários, melhorando seu visual em relação aos formulários HTML padrão.

form-check-label & form-check-input:

  • Na documentação é sugerido que cada grupo de componentes label + checkbox e label + radio sejam colocados agrupados em um único elemento de bloco (div) com margem inferior de 3 (mb-3).
  • A classe form-check-label estiliza os elementos <lable> e a classe form-check-input estiliza os elementos <checkbox> e <radio>.

checkbox & radio

(form-check-label & form-check-input)

form-switch:

  • A classe form-switch renderiza os elementos checkbox e radio com aparência de botões horizontais. Ela deve ser colocada no elemento de bloco que envolve os componentes.

checkbox & radio

(form-switch)

form-check-inline:

  • A classe form-check-inline renderiza os elementos checkbox e radio na horizontal. Ela deve ser colocada no elemento de bloco que envolve os componentes.

checkbox & radio

(form-check-inline)

form-check-reverse:

  • A classe form-check-reverse renderiza os elementos checkbox e radio do lado direito do componente pai. Ela deve ser colocada no elemento de bloco que envolve os componentes.

checkbox & radio

(form-check-reverse)

btn-check:

  • A classe btn-check renderiza os elementos checkbox e radio como botão marcável, ou seja, quando clicado torna o controle mais escuro. A classe de cores (btn-[cores]) deve ser colocada no label e ter o id do input com o mesmo valor no for da label.

Atributos do elemento checkbox & radio:

  • Alguns atributos HTML dos elementos têm sua própria renderização no Bootstrap.

disabled

  • O atributo disabled torna o controle desabilitado.

checkbox

checkbox (outline)

radio

radio (outline)

name="" & value=""

  • Os atributos name & value trabalham em conjunto. No checkbox cada elemento terá um name diferente representando cada escolha, já que geralmente tem mais de uma opção e pode ser feita mais de uma escolha, e um value que representa o valor do campo escolhido que será enviado ao servidor para tratamento ou guarda no Banco de Dados. No radio o name é único em todas as opções para poder escolher somente uma delas, desmarcando outra que tenha sido marcada dentro das opções fornecidas e cada uma com um value diferente com o valor que será enviado ao Banco de Dados para tratamento ou guarda.

Sistema de Layouts em GRID

  • O sistema de GRID do Bootstrap 5 usa o flexbox - CSS e elementos de bloco com as classes .container (centraliza o container na página de forma responsiva), .row e .col que possibilitam criar layouts de colunas e alinhar o conteúdo de forma totalmente responsiva.

classe .col sem especificação de colunas:

  • Como o Bootstrap trabalha com a divisão virtual do container .row em 12 colunas, sempre que os elementos da classe .col não tiverem especificado a quantidade de colunas, o Bootstrap agrupará cada elemento com a mesma largura, preenchendo todo o elemento da classe .row, independente da quantidade de elementos da classe .col.

classe .col com especificação de colunas:

  • Sempre que todos os elementos da classe .col tiverem a especificação da quantidade de colunas, o Bootstrap agrupará cada elemento com a largura específica da quantidade de colunas conforme o indicado na classe .col. Não podendo ultrapassar as 12 colunas virtuais.

classe .col com especificação de colunas maior que 12 colunas:

  • Caso os elementos da classe .col tiverem a especificação da quantidade de colunas que somadas superem as 12 colunas, o Bootstrap criará uma linha virtual empurrando os últimos elementos para baixo.

classe .col com e sem especificação de colunas:

  • Quando um ou mais elementos da classe .col tiver especificado uma quantidade de colunas e houver mais elementos da classe .col no container .row basta deixar os demais elementos .col sem especificação e o Bootstrap preencherá o restante do container .row com a divisão exata na largura dos elementos restantes da classe .col.

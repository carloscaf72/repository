Posicionamento

  • O Bootstrap disponibiliza várias classes que trabalham o Posicionamento dos Componentes.

Classes fixed-top & fixed-bottom:

  • A classe fixed-top posiciona o elemento no topo da viewport e a classe fixed-bottom posiciona o elemento na base da viewport.
  • Ambas as classes retiram o elemento do fluxo do documento como se estivessem flutuando na página e isso faz com que os demais elementos no entorno passem por baixo, podendo ficar parcial ou totalmente encobertos. exemplo

Classes sticky-top & sticky-bottom:

  • A classe sticky-top posiciona o elemento no topo da viewport e a classe sticky-bottom posiciona o elemento na base da viewport após a rolagem da página, ou seja, o elemento somente ficará fixo no topo ou na base quando alcançar a posição estabelicida nas classes.
  • As classes não retiram o elemento do fluxo do documento e somente os deixam fixos nas posições estabelecidas após rolagem da página. exemplo

Classe vstack:

  • A classe vstack cria uma pilha vertical sendo possível aplicar a classe gap para o espaçamento entre os elementos.

Classe hstack:

  • A classe hstack cria uma pilha horizontal transformando o elemento em um tipo de flexbox e seus filhos em elementos inline-block.
  • É possível no elemento com a classe hstack colocar uma linha vertical com um elemento com a classe vr.

Classe position:

  • A classe positon trabalha com as variações absolute, fixed, relative, static e sticky.
  • A classe positon-absolute retira o elemento do fluxo do documento, sendo possível determinar sua posição com as classes de direção: top, end, bottom e start + traço (-) + numeros: 0 - 50 - 100 - que representam a porcentagem em relação à borda da página.
  • Esta posição será em relação ao elemento de primeira hierarquia mais alta (pai) que estiver com alguma position que não seja a static (padrão).

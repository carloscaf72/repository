Formulários

  • O Bootstrap contém diversas classes para formatar formulários, melhorando seu visual em relação aos formulários HTML padrão.

form-label & form-control:

  • Na documentação é sugerido que cada grupo de componentes label + input sejam colocados agrupados em um único elemento de bloco (div) com margem inferior de 3 (mb-3).
  • A classe form-label estiliza os elementos <lable> e a classe form-control estiliza alguns elementos <input>.
  • Por padrão, a classe form-control transforma os inputs em elementos de bloco, começando na próxima linha e ocupando 100% da largura do elemento pai.

form-text:

  • Para colocar uma descrição do campo ou alguma orientação ao usuário, coloca-se a classe form-text dentro de um elemento de bloco e sua aparência será de um texto menor com cor suave.

Atributos do elemento <input>:

  • Alguns atributos HTML dos elementos têm sua própria renderização no Bootstrap.

value=""

  • O atributo value é opcional no <input> e coloca um valor inicial no controle permitindo ser alterado pelo usuário.

disabled

  • O atributo disabled torna o controle desabilitado.

readonly

  • O atributo readonly torna o controle somente leitura.

placeholder=""

  • O atributo placeholder permite colocar um texto de exemplo ou orientação de escrita no controle que não servirá como valor.

readonly + form-control-plaintext

    • O atributo readonly torna o controle somente leitura e a classe form-control-plaintext retira a aparência do controle, tornando-o um texto comum. Utilizado nos casos onde os dados não devam ser alterados.

form-control-sm & form-control-lg:

  • Os tamanhos dos elementos <input> podem ser alterados com com a classe form-control, acrescentada dos sufixos sm e lg (form-control-sm, form-control-lg).

<datalist>:

  • O elemento <datalist> é utlizado para criar uma lista de opções que pode alimentar o controle <input> type="text". Ela é montada com os elementos <option> e associada ao controle através do atributo list que terá o mesmo valor do id do datalist, criando a lista de opções no controle input.

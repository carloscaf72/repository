Visibilidade

  • O Bootstrap disponibiliza várias classes que trabalham a Visibilidade dos Componentes.

Classe d-block (display-block):

  • A classe d-block transforma qualquer elemento em elemento de bloco, portanto, por padrão, todos os elementos de bloco vêm com esta classe e assim como no HTML ocuparão toda a largura do componente Pai e começarão na próxima linha.
<section>: d-block

<p>: d-block

<div>: d-block
<span>: d-block

Classe d-inline (display-inline):

  • A classe d-inline transforma qualquer elemento em elemento de linha, portanto, por padrão, todos os elementos de linha vêm com esta classe e assim como no HTML ocuparão altura e largura somente do seu conteúdo e não aceitarão configuração de elementos de bloco.
<section>: d-block

<p>: d-inline

<div>: d-inline
<span>: d-inline

Classe d-inline-block (display-inline-block):

  • A classe d-inline-block transforma qualquer elemento em elemento de linha, mas mantém a possibilidade de algumas configurações de um elemento de bloco.
<section>: d-block

<p>: d-inline-block

<div>: d-inline-block
<span>: d-inline-block

Classe d com breakpoints:

  • A classe d utiliza todas as variantes de responsividade com os sufixos de breakpoints, podendo criar layouts responsivos alterando inclusive a visibilidade do componente em diferentes resoluções.

div 01: d-none d-md-block

div 02: d-none d-md-block

div 03: d-sm-none d-xl-block

Classe d para impressão:

  • A classe d utiliza configurações para impressão, podendo retirar e colocar elementos na impressão, assim como na tela (viewport).

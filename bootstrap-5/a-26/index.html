  • O Bootstrap contém diversas classes para criar barras de navegação, melhorando seu visual em relação ao HTML padrão.

btn-group:

  • Na documentação é sugerido que cada grupo de controles sejam colocados agrupados em um único elemento de bloco (div) com margem inferior de 3 (mb-3).
  • A classe btn-group é usada para agrupar botões (<input> (submit ou reset), <radio>, <checkbox>, etc) quando colocados dentro do elemento de bloco (div) que contém os controles.

    collapse:

    justify-content-center:

    • A classe collapse possibilita mostrar e esconder elementos a partir de um click de botão ou de um link.
    • Para isso é criado um elemento com a TAG <button> ou <a> (link), ambos com o atributo data-bs-toggle com o valor collapse (data-bs-toggle="collapse") e com a classe btn.
    • Ao elemento da TAG <button> é acrescentado o atributo data-bs-target com um valor apontado para o id do elemento com a classe collapse (data-bs-target="#id-collapse") e o elemento da TAG <a> (link) será apontado o id do elemento com a classe collapse através do atributo href (href="#id-collapse").

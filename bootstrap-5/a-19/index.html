  • O Bootstrap contém diversas classes para formatar formulários, melhorando seu visual em relação aos formulários HTML padrão.

form-select:

  • Na documentação é sugerido que cada grupo de componentes label + select sejam colocados agrupados em um único elemento de bloco (div) com margem inferior de 3 (mb-3).
  • A classe form-label estiliza os elementos <lable> e a classe form-select estiliza os elementos <select>.
  • Por padrão, a classe form-select transforma os selects em elementos de bloco, começando na próxima linha e ocupando 100% da largura do elemento pai.

<select> formatado por form-select

form-select-sm & form-select-lg:

  • Os tamanhos dos elementos <select> podem ser alterados com a classe form-select, acrescentada dos sufixos sm e lg (form-select-sm, form-select-lg).

Atributos do elemento <select>:

  • Alguns atributos HTML dos elementos têm sua própria renderização no Bootstrap.

name="" & value=""

  • Os atributos name & value trabalham em conjunto. O <select> terá um name que representa o nome do campo e cada elemento <option> terá um value diferente representando a escolha do usuário, Caso o controle esteja com o atributo multiple, cada valor escolhido será enviado com o nome do campo colocado no <select>.

disabled

  • O atributo disabled torna o controle desabilitado.

multiple

  • O atributo multiple retira a aparência de combobox do controle, transformando-o em uma lista de opções de 4 unidades aparentes e com scroll lateral.

size=""

  • O atributo size retira a aparência de combobox do controle e define a quantidade de elementos <option> que ficarão aparentes.

form-range:

  • Na documentação é sugerido que cada grupo de componentes label + range sejam colocados agrupados em um único elemento de bloco (div) com margem inferior de 3 (mb-3).
  • A classe form-label estiliza os elementos <lable> e a classe form-range estiliza os elementos <range>.
  • Por padrão, a classe form-range transforma os ranges em elementos de bloco, começando na próxima linha e ocupando 100% da largura do elemento pai.

<range> formatado por form-range

Atributos do elemento <range>:

  • Alguns atributos HTML dos elementos têm sua própria renderização no Bootstrap.

min="" & max=""

  • O atributo min e max são opcionais no elemento <range> e definem, respectivamente, um valor mínimo e um valor máximo para o componete.

name="" & value=""

  • Os atributos name & value trabalham em conjunto. O <range> terá um name que representa o nome do campo e seu valor será preenchido pela escolha do usuário. Se for colocado um value, o campo iniciará com este valor, indicando em que altura o seletor ficará na renderização. Caso o usuário não faça nenhuma escolha, seá o que está no value o valor enviado ao Banco de Dados para tratamento ou guarda.

step=""

  • O atributo step é opcional no elemento <range> e define o intervalo de passos entre os valores do componente. Por padrão este intervalo é de 1 unidade.

disabled

  • O atributo disabled torna o controle desabilitado.

btn:

  • A classe btn configura os elementos com aspecto de um botão clicável, sendo necessário acrecentar outras classes para formatações de cores e layouts.

btn-[cores]

  • Para configurar as cores do botão, acrescentamos a classe btn + traço (-) + cores temáticas do bootstrap.
link

btn-outline-[cores]

  • Para configurar os botões somente com cores nas bordas, acrescentamos a classe btn + traço (-) + outline + traço (-) + cores temáticas do bootstrap.
link

btn-sm & btn-lg:

  • Os tamanhos dos elementos com a classe btn podem ser alterados com a classe btn acrescentada dos sufixos sm e lg (btn-sm, btn-lg).

Atributos da classe btn:

  • Alguns atributos HTML dos elementos têm sua própria renderização no Bootstrap.

disabled

  • O atributo disabled torna o controle desabilitado. Não funcionando nos elementos a (link) e label.

data-bs-toggle

  • O atributo data-bs-toggle é um atributo especial do bootstrap que torna o botão marcável, ou seja, quando clicado torna o controle mais escuro.

Referência via Arquivo Interno

  • O Bootstrap é gratuito e permite que seus arquivos sejam baixados e utilizados mesmo que comercialmente, sendo possível referenciá-lo no documento HTML com uma TAG <link> para o CSS e uma TAG <script> para o JS aos arquivos que estarão em diretórios específicos dentro do computador/servidor do ducumento HTML.
  • Referenciando CSS na TAG <link> via Arquivo Interno:
    <link rel="stylesheet" href="./css/bootstrap/bootstrap-5.2.3/bootstrap.min.css">
  • Referenciando JS na TAG <script>via Arquivo Interno:
    <script src="./js/bootstrap/bootstrap-5.2.3/bootstrap.bundle.min.js">
  • Obs.: Na referência atravérs de Arquivos Internos não ocorrerá atualização dos arquivos se novas versões não forem baixadas para os diretórios, portanto, não ocorrerá problemas de alteração de versão.

Painéis Deslizantes

  • O Bootstrap contém diversas classes para criação e configurações de canvas, melhorando seu visual em relação ao HTML padrão.

a - offcanvas:

offcanvas-position / show / tabindex="-1":

  • O componente offcanvas cria um painel deslizante onde podemos colocar qualquer elemento HTML (menu, image, links, etc).
  • Para a criação de um offcanvas utilizamos como componente principal um elemento de bloco (div) com a classe offcanvas e a classe offcanvas + traço (-) + a configuração da posição de abertura do painel: start (à esquerda), top (superior), end (à direita) e bottom (inferior).
  • É obrigatório a colocação do atributo tabindex="-1".
  • É necessário a referência ao JS do bootstrap (bootstrap.bundle.min.js).
  • A classe show possibilita a abertura no carregamento da página.
Exemplo

b - offcanvas:

offcanvas-header & offcanvas-body:

  • Podemos dividir as regiões do painel em offcanvas-header (cabeçalho do painel) e offcanvas-body (corpo do painel).
  • No offcanvas-header podemos inserir um título com uma TAG de texto e a classe offcanvas-title e um botão de fechamento com a TAG button com a classe btn-close e o atributo data-bs-desmiss com o valor offcanvas (data-bs-desmiss="offcanvas").
Exemplo

c - offcanvas:

nav-bar & button:

  • Podemos colocar um elemento de botão <button> ou <a> em um componente nav-bar para abertura e fechamento do painel. Para isso criamos o componente nav-bar e no elemento <button> colocamos o atributo data-bs-toggle com o valor offcanvas (data-bs-toggle="offcanvas") e o atributo data-bs-target com o valor apontando para o id do componente offcanvas e no elemento <a> colocamos o atributo data-bs-toggle com o valor offcanvas (data-bs-toggle="offcanvas") e no atributo href apontando para o id do componente offcanvas.
Exemplo

d - offcanvas:

data-bs-scroll:

  • Para podermos usar o scroll do mouse mesmo com o painel offcanvas aberto, basta colocarmos no componente principal (offcanvas) o atributo data-bs-scroll com o valor true. (data-bs-scroll="true")
Exemplo

e - offcanvas:

data-bs-backdrop:

  • Para configurarmos o backdrop, que é a parte restante da página que escurece com a abertura do painel, podemos utilizar o atributo data-bs-backdrop.
  • O atributo data-bs-backdrop com o valor false. (data-bs-backdrop="false") retira o escurecido e impede que seja clicado fora do componente e que o mesmo retraia.
  • O atributo data-bs-backdrop com o valor static. (data-bs-backdrop="false") não retira o escurecido, mas impede que seja clicado fora do componente e que o mesmo retraia, impedindo também que qualquer outro componente seja clicado.
Exemplo

f - offcanvas:

colors:

  • Para configurarmos a cor do painel offcanvas podemos utilizar as classes de cores temáticas do bootstrap.
Exemplo

g - offcanvas:

--bs-offcanvas-width & --bs-offcanvas-height:

  • Podemos configurar a largura dos painéis com o position start e end e a altura dos painéis top e bottom do offcanvas utilizando uma TAG <style> no componente principal (offcanvas), configurando a variável --bs-offcanvas-width para a largura e --bs-offcanvas-heigth para a altura, podendo utilizar medidas em porcentagem ou pixels.

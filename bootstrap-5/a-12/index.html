Icons

  • O Booptstrap tem uma biblioteca de código aberto e de alta qualidade com mais de 1800 ícones da extensão .SVG disponibilizados para o uso particular ou comercial totalmente gratuito.

Baixando os Ícones via npm:

  • é possível baixar os ícones do Bootstrap no site icons.getbootstrap.com e utilizá-lo como quiser ou fazer a instalação no projeto via npm (node).

Utilização dos Ícones via CDN:

  • a melhor utilização dos ícones do Bootstrap é através do CDN: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.3/font/bootstrap-icons.css">. Assim será possível aproveitar todas as formatações de mudança de cores e tamanhos e diminuindo o fluxo de imagens do servidor de hospedagem.
  • para colocar o ícone na página é utilizado a TAG <i> com a classe bi + traço (-) + o nome do ícone que pode ser verificado na coleção de ícones da página icons.getbootstrap.com.
  • <i classe="bi-7-circle">
  • <i classe="bi-7-circle-fill">
  • <i classe="bi-cloud-check">
  • <i classe="bi-cloud-check-fill">

Formatação do Ícones (Tamanho e Cores):

  • os ícones Bootstrap são vetoriais, portanto, a formatação para alteração do tamanho e da cor de dará através da manipulação de formatação de textos, ou seja, alterar o tamanho e a cor do texto fará com que os ícones sejam formatados como desejado.
  • pode ser utilizado classes do próprio Bootstrap (fs-1) ou colocado em um atributo style na TAG <i> ou pode-se criar uma TAG <style> no cabeçalho do documento (<head>) ou em um outro arquivo CSS vinculado à página.
  • <i classe="bi-7-circle fs-1">
  • <i classe="bi-7-circle-fill" style="font-size: 40px; color: blue;">
  • <i classe="bi-cloud-check">
  • <style>
    i {font-size: 64px;
    }
    </style>

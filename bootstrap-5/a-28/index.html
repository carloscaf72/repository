Alerts-Badges-List

  • O Bootstrap contém diversas classes para criação e configurações de alerts, badges e lists, melhorando seu visual em relação ao HTML padrão.

alert:

  • Para a criação de um alert utilizamos um elemento de bloco (div) com a classe alert e a classe alert + traço (-) + uma das cores temáticas (primary, secondary, success, danger, info, warning, dark, light).
  • Por padrão, o componente ocupará 100% da largura do elemento pai.

<div>

(alert alert-info)

Este é um alerta simples

alert-link:

  • Para incluirmos um link ao alert o criamos com a classe alert-link.

<div class="alert">

<a class="alert-link">

Este é um alerta simples

saiba mais...

alert-heading:

  • Para incluirmos um Título ao alert o criamos com a classe alert-heading, podendo inclusive acrescentar ícones.
  • A classe alert-heading não altera a aparência dos componentes, devendo assim ser complementada com elementos ou classes para formatação do tamanho e/ou aparência.

<div class="alert">

<h3 class="alert-heading">

<i class="bi-info-circle">

Informação

Este é um alerta simples

saiba mais...

alert-dismissible:

  • Podemos incluir um botão de fechamento ao alert acrescentando a classe alert-dismissible ao componente e incluindo um <button> logo após o conteúdo do alert com a classe btn-close e o atributo data-bs-dismiss com o valor alert (data-bs-dismiss="alert").
  • Acrescentando as classes fade e show, tornamos o fechamento do alert mais suave.

<div class="alert alert-dismissible fade">

<button class="btn-close" data-bs-dismiss="alert">

Informação

badges:

  • A classe badges é utilizada para demonstrar pequenos conteúdos de forma destacada.

<h3>

<span class="badge">

Texto em um H3: novo novo

<button> badges:

  • A classe badges utilizada em um componente <button>.

<button class="btn btn-primary">

<span class="badge text-bg-danger">

list-group & list-group-item:

  • A classe list-group é utilizada para criar um container de agrupamento de listas.
  • A classe list-group-item é utilizada em cada um dos elementos que serão agrupados como numa lista.
  • A classe list-group-item-action é utilizada em cada um dos elementos para mudar a aparência ao passar o mouse.

<div class="list-group">

<li class="list-group-item">

  • item de lista 1
  • item de lista 2
  • item de lista 3
  • item de lista 4
  • item de lista 5

    • <div class="list-group">

    <a class="list-group-item list-group-item-action">

    link 1 link 2 link 3 link 4 link 5

    <div class="list-group">

    <button class="list-group-item list-group-item-action">

    list-group-flush

    • A classe list-group-flush deixa a aparência da lista mais suave, retirando as linhas de contorno.

    <div class="list-group-flush">

  • item 1
  • item 2
  • item 3
  • item 4
  • item 5

    • active & disabled:

    • A classe active deixa a aparência do elemento como ativo ou selecionado. A classe disabled deixa a aparência do elemento como inativo ou cancelado.

    <div class="list-group">

    <li class="list-group-item active">

    <li class="list-group-item disabled">

    list-group-numbered:

    • A classe list-group-numbered acrescenta uma numeração à lista mesmo em componentes que não sejam criadas com a TAG <ol>.

    <ol class="list-group-numbered">

    <li class="list-group-item">

    1. Laranja
    2. Mamão
    3. Maçã
    4. Pêra
    5. Uva

    <div class="list-group-numbered">

    <li class="list-group-item">

  • Laranja
  • Mamão
  • Maçã
  • Pêra
  • Uva

    • list-group & links completos:

    • Podemos utilizar a classe list-group para formar listas de elementos como os cards, clicáveis, com a TAG <a>.

    <div class="list-group">

    <a class="list-group-item list-group-item-action">

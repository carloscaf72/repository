Layouts de Formulários

  • O Bootstrap contém diversas classes para formatar formulários, melhorando seu visual em relação aos formulários HTML padrão.

layout:

  • Na documentação é sugerido que cada grupo de componentes label + input sejam colocados agrupados em um único elemento de bloco (div) com margem inferior de 3 (mb-3). Com isso, o padrão dos formulários será de um elemento abaixo do outro formando uma pilha.
  • Para colocarmos mais de um elemento numa linha recorremos ao sistema das classes row e col.

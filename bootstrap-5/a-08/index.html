Tipologia

  • O Bootstrap renderiza a página com uma fonte padrão sem serifa sans-serif.

Renderizando elementos de Título (<h1>, <h2>, <h3>, <h4>, <h5>, <h6>):

  • Os elementos de Título <h1> a <h6> são renderizados acompanhando a fonte padrão dos demais itens de texto da página, porém, as características de tamanho são similares ao do HTML comum.

h2 - Título de Nível 02

h3 - Título de Nível 03

h4 - Título de Nível 04

h5 - Título de Nível 05
h6 - Título de Nível 06

Deixando outros elementos como aparência de Títulos:

  • Outros elementos de Texto podem utilizar as classes .h1 a .h6 para que a formatação seja similar aos elementos de título, porém, sem a semântica do HTML5.

Aparência de Títulos mais destacados para qualquer TAG com a classe .display:

  • A classe .display destaca elementos de título assim como os demais elementos de texto, porém, sem a semântica do HTML5.

Podemos destacar textos com a classe .lead:

  • A classe .lead destaca elementos de título assim como demais elementos de texto, porém, sem a semântica do HTML5.

TAGs HTML para texto e como o Bootstrap renderiza:

  • As classes para formatação de textos são renderizadas pelo Bootstrap com a fonte padrão.

TAGs com semântica:

Texto marcado - TAG <mark>

Texto pequeno - TAG <small>

Texto superior - TAG <sup>

Texto inferior - TAG <sub>

Texto excluído - TAG <del>

Texto recém-inserido - TAG <ins>

Texto encorpado - TAG <strong>

Texto com ênfase - TAG <em>


TAGs sem semântica:

Texto riscado - TAG <s>

Texto sublinhado - TAG <u>

Texto com negrito - TAG <b>

Texto em itálico - TAG <i>

TAG <abbr> para acrônimos e abreviações:

  • Ao passar o mouse em cima da abreviação será mostrado o significado do texto que deve ficar dentro do atributo title. Esse atributo pode ser colocado em outras TAGs e terá o mesmo efeito, porém, sem a formatação da TAG abbr (sublinhado em forma de pontilhado e com o cursor com ?).
MERN - TAG abbr
CBF - TAG span

TAG <blockquote> para citações longas:

  • A TAG blockquote tem a semântica de uma citação de terceiros, podendo acrescentar uma TAG cite para que possa colocar o autor da citação.

Visual padrão do Bootstrap para a TAG <blockquote> (citação). Blá, blá, blá, blá, blá, blá, blá, blá, blá, blá, blá, blá.

Containers Responsivos

  • A classe .container é uma das mais importantes para a construção de layouts responsivos usando Bootstrap 5. Ela permite criar uma região de conteúdo centralizado na página com uma largura fixa para cada resolução de tela ou de largura variável ocupando toda a largura disponível, sempre com uma margem interna (padding) pré-definida.
  • A documentação do Bootstrap define três (3) tipos de Containers: .container, .container-fluid e .container-{breakpoints} que seria a classe container com cada um dos infixos de breakpoints (sm-md-lg-xl-xxl).
  • A tabela abaixo demonstra como o elemento de bloco deverá se comportar nas diferentes classes .container em relação ao tamanho da resolução de tela, sendo que sempre estará centralizado na tela.

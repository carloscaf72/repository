Carousel

  • O Bootstrap contém diversas classes para criação e configurações de carousel, melhorando seu visual em relação ao HTML padrão.

carousel & slide:

  • Para a criação de um carousel utilizamos como componente principal um elemento de bloco (div) com a classe carousel e a classe slide, acrescentando o atributo data-bs-ride com o valor carousel (data-bs-ride="carousel").

carousel-inner:

  • Dentro do componente principal utilizamos mais um elemento de bloco (div) com a classe carousel-inner que será responsável pelos compnentes com as imagens (slides).

carousel-item:

  • Cada elemento de bloco (div) da classe carousel-item representa um slide da apresentação. Esses componentes serão inseridos dentro do elemento da classe carousel-inner, sendo necessário colocar a classe active no 1º elemento com imagem (slide).

carousel-control-prev:

  • A classe carousel-control-prev é colocada em um componente <button>, abaixo do componente (carousel-inner), com o atributo data-bs-slide com o valor prev (data-bs-slide="prev") e o atributo data-bs-target com o valor apontando para o id do componente principal (carousel slide) (data-bs-target="#id-slide"), sendo colocado como conteúdo do <button> um elemento <span> com a classe carousel-control-prev-icon e serve para acessar o slide anterior ao apresentado.

carousel-control-next:

  • A classe carousel-control-next é colocada em um componente <button>, abaixo do componente (carousel-inner), com o atributo data-bs-slide com o valor next (data-bs-slide="next") e o atributo data-bs-target com o valor apontando para o id do componente principal (carousel slide) (data-bs-target="#id-slide"), sendo colocado como conteúdo do <button> um elemento <span> com a classe carousel-control-next-icon e serve para acessar o slide posterior ao apresentado.

carousel-indicators:

  • Para acrescentarmos indicadores aos slides criamos um elemento de bloco (div), logo acima do componente (carousel-inner), onde serão colocados componentes da TAG <button> com o atributo data-bs-target com o valor apontando para o id do componente principal (carousel slide) (data-bs-target="#id-slide") e o atributo data-bs-slide-to com o valor que será um número indicando o indice do slide (data-bs-slide-to="0").
  • É necessário que a quantidade de componentes da TAG <button> seja igual a quantidade de slides da apresentação, sendo necessário colocar a classe active no 1º elemento assim como foi feito nos slides (carousel-item).

carousel-caption:

  • Para acrescentarmos textos aos slides criamos um elemento de bloco (div) dentro do componente carousel-item com a classe carousel-caption, colocado logo abaixo da TAG <img>.
  • A classe carousel-caption não altera a aparência dos componentes, devendo assim ser complementada com elementos ou classes para formatação do tamanho e/ou aparência.

carousel-fade:

  • Para alterarmos a apresentação dos slides de deslizante para esmaecendo, inserimos a classe carousel-fade ao componente principal (carousel slide).

data-bs-interval:

  • Podemos escolher o tempo de apresentação dos slides acrescentando a cada slide (carousel-item) o atributo data-bs-interval com o valor sendo colocado em milissegundos (data-bs-interval="3000").

data-bs-touch:

  • Podemos desabilitar a função touch da apresentação dos slides, quando acesso for por celulares ou tablets acrescentando ao componente principal (carousel slide) o atributo data-bs-touch com o valor false (data-bs-touch="false").

carousel-dark:

  • Podemos alterar as cores dos botões de seleção, assim os textos acrescentando ao componente principal (carousel slide) a classe carousel-dark.
  • A classe carousel-dark altera todos os componentes dentro de cada slide, porém, os elementos de textos da classe carousel-caption poderão receber configurações com as classes de cores temática do bootstrap.

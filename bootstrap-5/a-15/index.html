Formatação de Textos

  • O Bootstrap disponibiliza várias classes para a formatação de textos.

Configurando o alinhamento dos textos:

  • Para alinhamento dos textos utilizamos a classe text + traço (-) + posição: start (à esquerda) [padrão], center (centro) e end (à direita).
text-start
text-center
text-end

Configurando o alinhamento dos textos em diferentes resoluções de largura:

  • Para alinhamento dos textos em diferentes resoluções de largura utilizamos a classe text + traço (-) + infixos breakpoints (sm, md, lg, xl, xxl) + posição: start (à esquerda), center (centro) e end (à direita).
text-center text-md-end text-xl-start
centralizado até 767px
à direita (end) a partir de 768px
à esquerda (start) a partir de 1200px

Configurando a quebra de linha dos textos:

  • Para retirar a quebra de linha dos textos utilizamos a classe text + traço (-) + nowrap.
text-nowrap

Configurando a quebra de linha em palavras:

  • Por padrão, palavras não são quebradas para outra linha, mas caso haja necessidade de que seja feito esta quebra, utilizamos a classe text + traço (-) + break.
text-break

Formatando as letras dos textos de diferentes apresentações:

  • Para mudarmos a apresentação das letras utilizamos as classes lowercase (todas minúsculas), uppercase (todas maiúsculas) e capitalize (todas iniciais maiúsculas).

Formatando as fontes dos textos de diferentes tamanhos:

  • Para mudarmos o tamanho das fontes utilizamos a classe fs + traço (-) + numeração de 1 a 6 que representa o mesmo tamanho dos Títulos (h1 a h6).
fs-1
fs-2
fs-3
fs-4
fs-5
fs-6

Formatando o peso o estilos das fontes:

  • Para mudarmos o peso das fontes utilizamos a classe fw + traço (-) + as palavras bold (negrito), semibold (semi-negrito), normal (normal) e light (leve). Para mudarmos o estilo das fontes utilizamos a classe fst + traço (-) + as palavras italic (itálico) e normal (normal).
texto com peso em negrito (fw-bold)
texto com peso em semi-negrito (fw-semibold)
texto com peso normal (fw-normal)
texto com peso leve (fw-light)
texto com estilo em itálico (fst-italic)
texto com estilo normal (fst-normal)

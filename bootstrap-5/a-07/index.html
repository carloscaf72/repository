Formatação de Colunas

  • Os elementos de bloco da classe .col dentro de um container da classe .row podem ser alinhados na Vertical ou na Horizontal conforme a necessidade e layout da página. Para isso o Bootstrap tem várias classes que possibilitam e facilitam este alinhamento.

Alinhamento Vertical

  • Para alinhamento Vertical no container da classe .row acrescentamos a classe align-items-start (alinhamento superior), align-items-center (alinhamento central) e align-items-end (alinhamento inferior).

  • O elemento da classe .row é um container com flexbox, portanto, os elementos da classe .col que estiverem fora deste container não serão distribuídos horizontalmente e se comportarão conforme um elemento de bloco empurrando o próximo elemento para a próxima linha.

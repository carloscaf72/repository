Bordas & Sombras

  • Para definir uma borda em um elemento utilizamos a classe border.

Configurando a cor da borda:

  • Para configurações de cores das bordas acrescentamos a classe border + traço (-) + cores bootstrap.
border border-primary
border border-secondary
border border-success
border border-danger
border border-warning
border border-info
border border-light
border border-dark
border border-white

Configurando a opacidade da cor da borda:

  • Para configurações de opacidade das cores das bordas acrescentamos a classe border + traço (-) + opacity + traço (-) + numeração: 10, 25, 50, 75, 100 que representa a porcentagem (%) de opacidade da borda.
border border-primary border-opacity-50
border border-secondary border-opacity-50
border border-success border-opacity-50
border border-danger border-opacity-50
border border-warning border-opacity-50
border border-info border-opacity-50
border border-light border-opacity-50
border border-dark border-opacity-50
border border-white border-opacity-50

Configurando a espessura da borda:

  • Para configurações de espessura das bordas acrescentamos a classe border + traço (-) + numeração de 0 a 5.
border border-0
border border-1
border border-2
border border-3
border border-4
border border-5

Configurando a posição da borda:

  • Para colocarmos bordas em diferentes posições do elemento alteramos a classe border acrescentando um traço (-) + posição: top (superior), end (à direita), bottom (inferior), start (à esquerda).
border-top (superior)
border-end (à direita)
border-bottom (inferior)
border-start (à esquerda)
border-start border-end (horizontal)
border-top bortder-bottom (vertical)

Configurando arredondamento dos cantos da borda:

  • Podemos arredondar os cantos das bordas utilizando a classe rounded.
border rounded
  • Podemos acrescentar um traço (-) + numeração de 0 a 5 para mudarmos a acentuação do arredondamento da borda.
border rounded-0
border rounded-1
border rounded-2
border rounded-3
border rounded-4
border rounded-5
  • Além da numeração, podemos utilizar um traço (-) + as palavras pill (pílula) e circle (círculo) para outros tipos de arredondamento da borda que, dependendo do tamanho do elemento formará diferentes formas.
rounded-pill
rounded-pill
rounded-circle
rounded-circle
  • Para colocarmos arredondamento em diferentes posições do elemento alteramos a classe rouded acrescentando um traço (-) + posição: top (superior), end (à direita), bottom (inferior), start (à esquerda). Porém, não será posível alterar a acentuação do arredondamento.
border rounded-top (superior)
border rounded-end (à direita)
border rounded-bottom (inferior)
border rounded-start (à esquerda)
  • Para colocarmos sombras nos elementos utilizamos a classe shadow acrescentada das classes shadows, shadows-sm ou shadows-lg.
shadow shadows
shadow shadows-sm
shadow shadows-lg
